The Auction app connects the general public with the inventory of abandoned items in a warehouse.

Features:

  • Admin users can create LOTS to sell one specific product;
  • Only another admin can approve the LOT;
  • Regular users can BID on LOTS with active status;
  • They can also see lots with expired and pending-approval status;
  • Users can search products by code or name;
  • They can also set lots as favourites;
  • Admins can block user´s registration numbers;
  • Admins and users can exchange messages in LOT page;
  • And more...

Employees of the medical lab can upload csv files with data from medical exams, which get stored in a postgres database (no ORM, every query was writen by hand in plain SQL) and then exposed in an API. The app users docker, sidekiq and has AJAX functionality to read from the API.

Features:

  • Upload csv files;
  • Display exams in JSON format through API;
  • Read the API and display exams;
